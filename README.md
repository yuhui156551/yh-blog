# blog
## 前后端分离博客项目

### 技术栈

  SpringBoot2、MybatisPlus、Redis、SpringSecurity、JWT、OSS、RabbitMQ、AspectJ、MySQL、Lombok、Knife4j、Vue、elasticsearch、WebSocket。
  
### 实现
  
  主要实现功能：登录校验、写博客、评论博客、上传图片、认证授权、更新浏览次数、后台权限控制、在线聊天室等。
  
  项目已部署，访问地址：www.bsyuhui.top
